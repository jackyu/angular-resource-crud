## Angular Resource CURD 練習小範例

理解 angularJS 的 ngResource 運作原理後，嘗試實作一個小範例讓自己更熟練，範例分成前、後端二部分的工作。

* 前端 (client): 負責主要的程式邏輯處理
* 後端 (server): 負責 api 的資料生成

### 執行方式

先啟動 server

```
$ node server\server.js
```

打開瀏覽器，輸入

```
localhost: 8080
```

即可看到執行畫面
